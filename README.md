### **Enemy  of  <font color=green>Animator</font> **

#### Parameters：

1. state : int 
2. attack : trigger
3. hit : trigger
4. skill : trigger
5. dead : bool

#### Layer：

0. Base Layer
1. Attack Layer
2. Dead Layer

使用`state`切换`layer`：

1. 当`state=1`时敌人正在`Move to Target`，`state=0`时正在等待`idle`动画结束，从而实现敌人到巡逻点后停滞一个idle动画的长度
2. 当`state=2`时转换到`Attack Layer`，此Layer权重为1，完全覆盖其他Layer动画
